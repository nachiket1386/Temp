Upload CSV File
Upload attendance data in CSV format. Maximum file size: 16MB.
CSV Format Requirements
Required Columns (in order):
  • EP number - Employee identifier
  • Name - Employee full name
  • Company - Company name
  • Plant - Plant/location
  • Department - Department name
  • Trade - Trade/skill category
  • Skill - Skill level
  • Date - DD-MM-YYYY format
  • IN1, OUT1 - First in/out times
  • IN2, OUT2 - Second in/out times
  • IN3, OUT3 - Third in/out times
  • Hours Worked - Decimal hours
  • Overtime - Overtime hours
  • Status - P, A, -0.5, or -1
Time Format:

Use HH:MM format (24-hour). Leave blank if not applicable.

Status Codes:
  • P Present
  • A Absent
  • -0.5 Half Day
  • -1 Full Day Deduction
Import Process
  1. Download the template to ensure correct format
  2. Fill in your attendance data
  3. First, upload with "Validate Only" to check for errors
  4. Fix any validation errors in your CSV
  5. Upload again with "Import Data" to save to database
Note: New employee accounts will be automatically created with EP number as username and password.
